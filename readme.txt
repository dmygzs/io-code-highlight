代码高亮插件，支持经典编辑器和古腾堡代码块扩展。

== Description ==

io Code Highlight 使用 enlighterjs 语法高亮显示支持。

安装启用即可使用，无需复杂的设置，支持经典编辑器和古腾堡代码块扩展。

== Installation ==

1.上传 `io-code-highlight`目录 到 `/wp-content/plugins/` 目录
2.激活插件。
